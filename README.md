# 8fish
detect and recognize 8 kinds fish using yolov3, UI is made by QT.
使用yolov3检测和识别8种鱼，用QT编写

config中的config.xml为配置文件。
rtsp项填入rtsp视频地址。
scale项为计算鱼获长度的配置项。

为保证识别效果，鱼获需要横着放置在视频中！
